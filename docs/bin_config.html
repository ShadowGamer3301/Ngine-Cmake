NCF files

NCF (Ngine configuration format) is a binary format created to hold configuration data for Ngine Core. Under any circumstances it should not be used as configuration file for game runtime, since Ngine does not provide any ability to read custom data from configuration file.

Little and Big endian

NCF files are optimized to be used with Little endian processors (most of the AMD64 processors). Currently processors using Big endian are not supported by built-in functions of Ngine and data has to be converted manually by reordering data in files.

Data

This table list all data held in NCF file

Name Offset (in bytes) Size (in bytes) Reccomended value type Description
WindowFullscreen 0 1 bool Marks if windows should be created in fullscreen mode. 0 for windowed mode, 1 for fullscreen mode
WindowWidth 1 4 uint32_t Sets window width in pixels. Any value above 0 is accepted.
WindowHeight 5 4 uint32_t Sets window height in pixels. Any value above 0 is accepted.
AutoPickDevice 9 1 bool Sets if GPU used for rendering will be specified by user or if Ngine should pick the most suitable one.
ManualDeviceIndex 10 2 uint16_t Marks which GPU should be used for rendering if AutoPickDevice is set to false. Counting starts from 0.
EnableGfxDebugMode 12 1 bool Marks if Vulkan validation layers or D3D11 debug mode should be switched on.
WindowResize 13 1 bool Marks if dynamic resizing for window should be allowed.

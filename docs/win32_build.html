Required hardware

To build for Windows you need physical computer or virutal machine running Windows 10 or newer.

Required software

This is list of all the software that you need to compile project for windows.

  • Windows SDK: Windows 10 SDK (10.0.14393.0)
  • Microsoft GDK: October 2024 public release
  • CMake: Version 3.12 or newer
  • Ninja: newest version possible
  • C++ compiler: MSVC v143

Building

Steps to build project for Xbox

  1. Open Ngine project as CMake project in Visual Studio 2022
  2. In configuration select "Win32 debug" or "Win32 release"
  3. Compile project

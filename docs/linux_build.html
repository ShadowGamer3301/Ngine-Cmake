Required hardware

To build for Linux you need physical computer or virutal machine running GNU/Linux (see compatibility list down below).

Required software

This is list of all the software that you need to compile project for Linux.

  • Vulkan SDK: Version 1.3 or newer
  • CMake: Version 3.12 or newer
  • Ninja: newest version possible
  • C++ compiler: LLVM Clang

Building

Steps to build project for Xbox

  1. Open Ngine project as CMake project in Visual Studio 2022
  2. In configuration select "Linux debug" or "Linux release"
  3. Compile project

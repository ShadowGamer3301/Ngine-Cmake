Required hardware

To build for Xbox devices, a Xbox Series console with activated DevMode is required. Although compiling for Xbox One consoles should be possible it has been never tested, and will likely never will. Creators of Ngine do not guarantee any compatibility with Xbox One S or Xbox One X consoles. Also to compile project for Xbox a computer or virtual machine with Windows 10 or newer is required.

Required software

This is list of all the software that you need to compile project for windows.

  • IDE: Microsoft Visual Studio 2022
  • Windows SDK: Windows 11 SDK (10.0.22000.0)
  • Microsoft GDK: October 2024 public release
  • CMake: Version 3.12 or newer
  • Ninja: newest version possible
  • C++ compiler: MSVC v143

Building

Steps to build project for Xbox

  1. Open Ngine project as CMake project in Visual Studio 2022
  2. In configuration select "Xbox debug" or "Xbox release"
  3. Compile project
  4. Create required XML files
  5. Transfer project to your Xbox console
